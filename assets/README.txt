Installation:
****
1. Unzip all contents of this zip inside a new folder
2. Run the appropiate TwitchMIDI version for your operating system
    - If you use Windows: Double click on "TwitchMIDI-win.exe"
    - If you use Linux: Open a terminal and run ./TwitchMIDI-linux
    - If you use Mac OS: Open a terminal and run ./TwitchMIDI-macos
3. Have fun!
